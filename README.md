# points

An example social media app, written in dart with Flutter and supabase as a backend

## Getting started

### To run the project:
1. First you must run:
 `flutter pub install`
2. Run the app with:
 `flutter run --dart-define=supabase_url=your_supabase_url --dart-define=supabase_key=your_supabase_key`
